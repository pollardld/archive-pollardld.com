So I wanted to try out the new webkit srcset, and it is below. The correct result is only available in the nightly build of safari, but will be running into all webkit browsers soon.

Heres what you need:

  • img tag
  • fallback src
  • srcset for different viewports (ex., 2x = double pixel density, 600w = 600 screen width)
<img src="sailors_320_600.jpg" srcset="sailors_640_600.jpg 2x, sailors.jpg 2.5x">
