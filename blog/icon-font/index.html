Creating your own icon font can be a fun and rewarding process. Icon fonts are used every where now, as they are the best option for iconography. Why?

  1. Many icons stored in one font
  2. High def screen ready, scalable and sharp
  3. Easy to change through css
  4. and i am sure there are more reason i can't think of right now

How to make an icon font

  1. Create your glyphs
  2. Convert, easiest to use a converter tool, such as iconvault
  3. Or you can convert it to code / tables, but that would take away the fun
  4. Add the font, css and html to your site
