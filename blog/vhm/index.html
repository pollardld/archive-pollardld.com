Styling using viewport lengths might save you a massive head ache one day! Viewport styles include vw, vh and vm units. Viewport lengths measure 1/100th of the screen, scale proportionally on viewport change, and are fucking amazing.

That means if you want something to be 40% of the screen tall, just use 40vh. If you want something to be 80% of the screen wide, just use 80vw. Using vm will set something to which ever is smaller (vh or vw).

The reason i find this fucking AMAZING is vh. No more using position absolute for heights. No more dicking around lining up an inline-block. No more float and float left on a float left that floats right... Yes!
